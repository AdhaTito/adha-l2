CSV → Images (base64) — Export ZIP

Unggah file CSV apapun (nama bebas). Pilih kolom yang berisi base64 image dan kolom ID (mis. NIK). Aplikasi ini akan membuat ZIP berisi file gambar yang dinamai "{ID}_KTP.jpg" & "{ID}_FR.jpg" (atau sesuai pilihan).

Jika CSV tidak punya header, kolom akan diberi nama col_0, col_1, ...
